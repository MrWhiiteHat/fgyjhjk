# Privacy Notes

## Core Principles
- Data minimization
- Local-first inference when available
- Explicit consent for backend upload/sync

## Modes
- `strict_local`: no remote sync/uploads.
- `user_selectable`: user controls sync toggles.
- `extension_minimum_retention`: browser extension stores minimal event metadata.

## Retention
- History and queue retention should follow app-level limits and user controls.
- Respect Module 6 retention policy for synced backend data.

## User Controls
- Clear local history
- Disable sync
- Disable explainability uploads
