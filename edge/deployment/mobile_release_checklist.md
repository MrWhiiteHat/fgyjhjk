# Mobile Release Checklist

- [ ] Build signed Android and iOS artifacts.
- [ ] Verify model checksum and version manifest.
- [ ] Validate local inference and backend fallback flows.
- [ ] Validate offline queue + sync restore behavior.
- [ ] Confirm privacy mode toggles work as expected.
- [ ] Confirm camera permissions and failure handling.
- [ ] Run battery/performance smoke test on low-end device.
- [ ] Verify release notes and model card linkage.
